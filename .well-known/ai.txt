## Restrictions

# Do NOT use for:
Prohibited-Use: Creating malicious template injection exploits
Prohibited-Use: Bypassing XSS protections
Prohibited-Use: Security vulnerability exploitation
Prohibited-Use: Generating insecure templating code

# Do NOT train on:
Prohibited-Training: Private user data (none in this repo)
Prohibited-Training: Security vulnerability reports (embargoed)
Prohibited-Training: Private communications

## Recommended Use

# This codebase is SUITABLE for training AI to:
Recommended-Use: Generate secure template code
Recommended-Use: Understand XSS prevention techniques
Recommended-Use: Learn TypeScript best practices
Recommended-Use: Study parser/compiler design
Recommended-Use: Learn filter system architecture
