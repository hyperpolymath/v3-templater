/* ARCHITECTURE */

Core Engine:
- Lexer: Tokenization with custom delimiters
- Parser: AST generation with expression support
- Compiler: AST to executable code transformation
- Runtime: Safe expression evaluation (no eval!)

Security:
- Auto-escaping by default (XSS prevention)
- Sandboxed filter execution
- SafeString for trusted content
- Strict mode for error detection

Performance:
- LRU template caching
- Optimized compilation
- ~500K ops/sec with caching
